# Agent UI

A full-stack application for agent management and control with a React + Tailwind CSS dashboard.

## Project Structure

```
/agentui
  ├── /frontend     → React + Vite UI application
  ├── /backend      → Future API server (Node/Express or similar)
  ├── /instructions → Project documentation and guidelines
  ├── CLAUDE.md      → Project instructions for Claude Code
  └── README.md      → This file
```

## Clarification
There are two codebases:
- Application codebase: The tool that our team is developing, using Claude Code.
- Client codebase: The app that the client or end user is requesting our tool to create, using Claude SDK, or other tools.


## Documentation

- `CLAUDE.md` - Project instructions for Claude Code
- `instructions/refactor.md` - Refactoring guidelines
- `instructions/playwright.md` - Playwright MCP tool usage
